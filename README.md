# Описание
Приложение погоды городов России(установить можно и другие города, изменив файл StaticData/Cities.json, главное чтобы данные о них имелись на WeatherAPI) на 3 дня, с почасовым прогнозом, прогнозом на день и прогнозом на сейчас.

***

Основано на данных WeatherAPI из бесплатного плана.

***

Основной город приложения - Хабаровск - можно изменить в настройках приложения StaticData/AppSettings.json.

***

# Важно!
Чтобы приложение работало, необходимо получить API-ключ на сайте WeatherAPI.com, а затем вставить его в StaticData/secret.json вместо "YOUR_API_KEY_ON_WeatherAPI.com".
